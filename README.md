# 💼 Professional Expense Dashboard

A modern, responsive Expense Tracker Web Application built using **HTML, CSS, JavaScript, and Chart.js**.

This project helps users track income and expenses with a clean financial dashboard interface.

## 📌 Features

- ✅ Add Income & Expense transactions
- ✅ Category selection (Food, Travel, Bills, etc.)
- ✅ Automatic date tracking
- ✅ Real-time Balance calculation
- ✅ Total Income & Expense summary cards
- ✅ Animated Spending Ratio progress bar
- ✅ Doughnut Chart (Income vs Expense)
- ✅ Bar Chart (Transaction breakdown)
- ✅ Recent Activity section
- ✅ Data stored using LocalStorage
- ✅ Fully responsive design
- ✅ Modern fintech-style UI

## 🌐 How to Deploy

### Option 1: GitHub Pages
1. Push project to GitHub
2. Go to Settings → Pages
3. Select branch: `main`
4. Save
5. Access your live link

### Option 2: Netlify
1. Go to https://netlify.com
2. Drag & drop project folder
3. Get instant live link
